# Kolibri OS App

App example written in Rust & ASM.

## How to build:
To build this app you need these tools:
- binutils (ar)
- [flat assembler (fasm)](https://flatassembler.net/)
- cargo & rustc
- [kld](https://github.com/DarkoreXOR/kld)

then run `cargo build`
